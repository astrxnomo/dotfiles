---
name: commit-and-push
description: Create a commit, push and good message
---

1. Ejecuta un `git status`.
2. Añade los cambios al stage.
3. Actualiza la versión del proyecto: si es un proyecto web, eleva la versión en `package.json` en base a las modificaciones.
4. Crea un commit con un mensaje corto y descriptivo. No uses `Co-Authored-By: Claude` ni `git commit --amend`.
5. Push a la rama actual.
